# Aplicación de Evaluación Ergonómica TMERT

Esta aplicación Streamlit permite cargar una matriz TMERT en formato Excel, realizar una evaluación interactiva de riesgos con el método ART y generar un informe técnico detallado en formato Word.

## Instrucciones

1.  **Instalar dependencias:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Ejecutar la aplicación:**
    ```bash
    streamlit run app.py
    ```

3.  **Colocar la plantilla:**
    Asegúrate de que tu plantilla Word (`plantilla_ART_evaluacion.docx`) se encuentre en la carpeta `templates`.
